

# 🎯 Wordle Clone – React + TypeScript

A simple and clean **Wordle-inspired game** built with **React** and **TypeScript**.  
This project was created to practice frontend development, state management, and UI logic.

---
## 🔴 Live Demo

Check out the live demo [HERE](https://mm-97.github.io/wordle/)

---

## 🚀 Features

- 🔤 Guess a 5-letter word
- 🎨 Real-time letter feedback (correct, present, absent)
- ⚡ Fast and responsive UI
- 🌍 Switch between English 🇬🇧 and Italian 🇮🇹 word dictionaries
- 🧠 Basic game logic similar to the original Wordle

---

## 🛠️ Tech Stack

- **React**
- **TypeScript**
- **CSS**
- **Vite**

---

## 📦 Installation

Clone the repository and run:

```bash 
docker compose up --build
```
